# HR-Analytics-Dashboard
An interactive Tableau dashboard designed to analyze workforce metrics, attrition trends, employee demographics, and job satisfaction patterns. This project transforms HR data into actionable insights that support strategic workforce planning and talent management decisions.

## Purpose

The HR Analytics Dashboard is a comprehensive workforce analysis solution built using Tableau to monitor employee count, attrition rate, demographic distribution, and job satisfaction metrics.
The dashboard helps HR managers and business leaders identify attrition drivers, evaluate department-level trends, and make data-driven workforce decisions.
